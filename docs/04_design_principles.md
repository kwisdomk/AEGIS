# AEGIS Design Principles

## 1. Determinism

Same system state produces the same diagnostic output.

---

## 2. Portability

Runs on any Windows machine without installation.

---

## 3. Transparency

All diagnostics are derived from observable system data.

---

## 4. Separation of Concerns

Collection, analysis, and comparison are independent stages.

---

## 5. Non-Invasive Design

AEGIS does not modify system state or apply fixes.

---

## 6. Reproducibility

Results can be repeated across machines and time.
