# AEGIS Glossary

## Baseline

A structured snapshot of system state at a specific point in time.

---

## Wake Lock

A system or process preventing sleep/hibernate mode.

---

## CPU Min State

Minimum allowed CPU frequency under power management settings.

---

## DRAM-less SSD

A storage drive without dedicated cache memory, typically slower under load.

---

## C-States

CPU low-power idle states used for energy efficiency.

---

## JSON Baseline

Standard output format used by AEGIS containing system telemetry.

---

## Dual-Channel Memory

A memory configuration using two or more RAM modules to double memory bandwidth.

---

## Delta Report

The output of a comparison between two baselines, showing what changed.
